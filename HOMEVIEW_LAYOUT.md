## User Interactions

### Click Song Card
- Plays the song immediately
- Sets queue to trending list
- Starts from clicked position

### Click Play Button
- Same as clicking card
- Visual feedback

### Click Heart Button
- Adds/removes from liked songs
- Toggles heart fill state

### Click Refresh Button
- Forces immediate data fetch
- Shows loading spinner
- Updates timestamp

### Auto-Refresh (Background)
- Runs every 60 seconds
- No user action needed
- Smooth UI updates
- Shows updated timestamp
