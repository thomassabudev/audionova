# Keyboard Shortcuts for Sidebar Quick Actions

## Quick Actions Shortcuts
| Key | Action | Description |
|-----|--------|-------------|
| Q | Add to Queue | Add selected songs to queue |
| L | Like/Unlike | Like or unlike selected songs |
| P | Add to Playlist | Add selected songs to quick playlist |
| Shift | Multi-select | Hold to enter multi-select mode |
| Esc | Exit Multi-select | Exit multi-select mode |
| Enter | Activate | Activate focused button or action |

## Multi-Select Mode Shortcuts
| Key | Action | Description |
|-----|--------|-------------|
| Q | Add to Queue | Add all selected songs to queue |
| L | Like/Unlike | Like or unlike all selected songs |
| P | Add to Playlist | Add all selected songs to quick playlist |
| D | Download | Download all selected songs |
| Esc | Exit Mode | Exit multi-select mode |
| Space | Select/Deselect | Select or deselect focused item |

## Drag and Drop Shortcuts
| Key Combination | Action | Description |
|-----------------|--------|-------------|
| Ctrl+Shift+Arrow | Move Item | Move selected item in drag area |
| Ctrl+Space | Add to Quick Actions | Add focused item to quick actions |
| Esc | Cancel Drag | Cancel current drag operation |

## General Navigation
| Key | Action | Description |
|-----|--------|-------------|
| Tab | Navigate | Move between quick action buttons |
| Shift+Tab | Navigate Back | Move to previous button |
| Arrow Keys | Navigate Grid | Move between buttons in grid layout |
| Enter/Space | Activate | Activate focused button |

## Accessibility Shortcuts
| Key | Action | Description |
|-----|--------|-------------|
| F1 | Help | Open keyboard shortcuts help modal |
| Ctrl+Alt+H | Help | Alternative help shortcut |
| Esc | Close | Close tooltips, modals, or menus |

## Mobile-Specific Shortcuts
| Gesture | Action | Description |
|---------|--------|-------------|
| Long Press | Show Tooltip | Show tooltip for button |
| Swipe Down | Open Quick Actions | Open quick actions drawer |
| Two-Finger Tap | Multi-select | Enter multi-select mode |
| Double Tap | Activate | Activate button (alternative to single tap) |

## Customization
Users can customize these shortcuts in the Settings > Accessibility section of the application.
